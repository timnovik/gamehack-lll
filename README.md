# Towers and magic
Аркадная игра, сочетающая в себе жанры tower defence и roguelike. Подбор башен и их расположение — неотъемлемая стратегия игры. Обычно ползучие твари пробегают через подобие лабиринта, что дает игроку возможность стратегического размещения башен. Игроку предлагается на выбор несколько пересанажей с разным набором скилов. В игре используется пиксильная графика.
